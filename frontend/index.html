Services

PRODUCER

Log Ingestion

Reads system auth logs or accepts HTTP POST events. Parses raw lines into structured security events and publishes to Kafka.

:3001
AGENT

Detection Agent

Tracks failed logins per IP in Redis with a 60s window. Fires a brute force alert at 5+ failures. Deduplicates alerts for 5 minutes.

Kafka consumer
AGENT

Anomaly Agent

Detects logins between 11PM–6AM UTC, first-time IPs for a user, and sudo privilege escalation attempts. Stores seen IPs for 24hrs.

Kafka consumer
AGENT

Enrichment Agent

Classifies every event: IP type (private/public/loopback), known threat IPs check, and login timing context. Publishes to enriched_logs.

Kafka consumer
RESPONSE

Response Agent

Automatically blocks attacker IPs in Redis for 1 hour on high-severity alerts. Maintains daily alert counters per rule for analytics.

Kafka consumer
DASHBOARD

Dashboard Service

Saves alerts to PostgreSQL with deduplication. Serves REST API. Pushes new alerts to all connected browsers via WebSocket in real time.
